极影全能影音盒（Windows 专属）
产品定位
对标 PotPlayer + 网易云 + 视频格式转换，纯净无广告本地影音全能播放器，解决 Windows 用户：播放器卡顿、格式不兼容、广告弹窗、视频转码、音效调节痛点。
目标人群：影视爱好者、追剧党、剪辑新手、普通家庭用户。
核心功能
全格式万能播放
支持几乎所有视频 / 音频格式，4K、蓝光、HDR、字幕自动匹配，解码能力拉满，低配电脑也流畅。
纯净无广告无弹窗
无开屏广告、无推荐信息流、无捆绑安装，纯本地播放，拒绝影音软件流氓套路。
一键音视频转码
视频压缩、格式转换、截取片段、裁剪画面、提取音频，傻瓜式操作，无需专业知识。
高级播放增强
倍速播放、画面滤镜、音效均衡器、画面置顶、记忆播放进度、局域网播放共享。
本地歌单管理
本地音乐归类、自建歌单、无损音质适配，替代臃肿音乐客户端。
差异化亮点
抛弃主流播放器广告和冗余直播推荐，只做本地影音核心功能，集播放 + 转码 + 剪辑片段于一体，是 Windows 用户装机必备级工具。
商业模式
永久免费纯净版，无广告；仅靠皮肤主题、高级音效插件小额付费，不打扰普通用户。
